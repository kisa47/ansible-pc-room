# Ansible Classroom Configuration 

Цель: Автоматизация развёртывания и настройки компьютерных класса с помощью Ansible.

## Описание
Данный репозиторий содержит готовые Ansible-плейбук для быстрой и удобной настройки компьютерного класса для предмета "Облачные технологии". Система включает полную автоматизацию процессов установки программного обеспечения, ограничений пользователей и защиты системы.

Операционные системы: Ubuntu, Debian

## Быстрый старт
Клонируйте репозиторий:
```bash
git clone https://github.com/kisa47/ansible-pc-room.git
cd ansible-pc-room
```

Запустите основную роль конфигурации:
```bash
ansible-playbook -i inventory.yaml main.yaml --ask-become-pass
```
